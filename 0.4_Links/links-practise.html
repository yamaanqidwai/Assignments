Creating Links

Links are created using the {a} element which has an attribute called href. The value of the href attribute is the page that you want people to go to when they click on the link.

Linking to another page

click {a href="https://xyz.com" target="_blank"} here {/a}

Explanation

attribute href is used to make the link clickable and be able to take the user to that page(pointed by the link)
attribute target is used to tell the browser where to open the page
_blank = open in new tab

Linking to element on same page

click {a href="#ID-of-element" target=""} here {/a}

Explanation

attribute href is used to make the link clickable and be able to take the user to that element(pointed by the link and id allocated to it)
In order to link to element on same page, give id to that element and use #IDname in href
target could be set to any 4 of the available options.

Something similiar could be done to link to specific part of another page

In order to link to element on different page, give id to that element and give path to the page and use #IDname in href after giving / after the path
Eg 1: https://xyz.com/#IDname
Eg 2: ./path/#IDname
target could be set to any 4 of the available options.

Absolute URL:

URL stands for Uniform Resource Locator. Every web page has its own URL.
An absolute URL starts with the domain name for that site, and can be followed by the path to a specific page.
If no page is specified, the site will display the homepage.

Example of Absolute URL:

Going from current page to another page stored in another directory but on the same machine.
https://./../parent-child/child-page2

Explanation of Example

./ = go to current directory.
../ = go back one directory /directory-name = go the named directory

Relative URL

Its path generally refers to a resource on the same machine as the current document.

Example of Relative URL:

https://example.com/parent-page1/child-page1/grandchild
